## 🚀 Projects That Matter

### 🎵 [VibeCheck — Emotion-Driven Music Player](https://github.com/Chetan142006/VibeCheck-Emotion-Music-Player)

Real-time emotion detection from webcam feed or uploaded images using **DeepFace + OpenCV**, mapped live to Spotify playlists via the Spotify API. Classifies 7 distinct emotion states with a Python + JavaScript frontend.

```
Stack  →  Python · DeepFace · OpenCV · Spotify API · JavaScript
```

---

### 🔒 PII Redaction Middleware for LLM APIs

A middleware layer that intercepts user input before it hits any LLM API call, automatically detecting and redacting sensitive personal data — names, emails, ID numbers — using **spaCy NLP pipelines**. Designed as a plug-and-play safety wrapper for any AI application.

```
Stack  →  Python · spaCy · NLP · REST API · Regex
```

---

### 📊 Smart Attendance Prediction & Advisory System

Predictive model that flags students at academic risk based on historical attendance patterns. Generates advisory reports and threshold alerts used by faculty as a live decision-support tool.

```
Stack  →  Python · Scikit-learn · Pandas · MySQL · Data Visualization
```

---

### ☁️ Cloud-Powered Serverless Portfolio (AWS)

Personal portfolio deployed entirely on AWS — **S3 static hosting + CloudFront CDN + Route 53 DNS + Lambda** backend functions. Zero servers, near-zero cost, globally distributed.

```
Stack  →  AWS S3 · CloudFront · Lambda · API Gateway · Route 53 · HTML/CSS/JS
```
